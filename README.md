# Bot Reserva Canchas Las Condes

Bot automatizado para reservar dos canchas simultáneamente usando 2 RUTs diferentes.

## Características
- ✅ 2 RUTs en paralelo
- ✅ No recarga página (mejor rendimiento)
- ✅ Polling dinámico (más agresivo cerca de 18:00)
- ✅ Notificaciones Telegram
- ✅ Screenshots de confirmación

## Instalación

1. Clona el repositorio
2. Configura `BOT_TOKEN` en Railway
3. Deploy en Railway
4. Usa `/config` para configurar
5. Usa `/auto` para ejecutar

## Variables de Entorno
- `BOT_TOKEN`: Token de Telegram
